# Blog app
Small blog app, here you can get latest posts and create new one. 
For developing of this app such tecnohologies where used:
- typescript
-  axios
- lodash
- react
- nextjs
- redux
- redux-saga
- styled-components
- eslint
- prettier


## Install it and run:

```bash
npm install
npm run dev
# or
yarn
yarn dev
```
## Deployed app is [here](https://blog-sigma-liart.vercel.app/)
